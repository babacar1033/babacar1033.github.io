Work Samples & Project Descriptions

Three artifacts from the AIML-501 portfolio — each grounded in real course requirements, with context, the challenge solved, and the chosen solution.

Artifact 1

ML Algorithm Types: Visual Framework

View Artifact →

Context

The first portfolio artifact required a structured, visually compelling reference for ML algorithm types — covering learning paradigms and application domains in a way accessible to both technical and non-technical audiences.

Challenge

13 algorithms across 2 learning paradigms and 4 application domains (tabular, vision, NLP, generative AI) needed to be organized, explained, and made interactive — without becoming an overwhelming wall of text or a static PDF.

Solution

Built an interactive HTML page with live filter buttons (by paradigm and domain), hover-effect algorithm cards, a taxonomy tree, a domain-by-domain breakdown, and a comparison table. Deployed to GitHub Pages.

HTML5 Tailwind CSS Vanilla JS GitHub Pages
Artifact 2

Neural Network Components: Visual Presentation

View Artifact →

Context

This artifact required defining the six core neural network components — layers, neurons, weights, activation functions, loss functions, and optimization algorithms — grounded in hands-on exploration with TensorFlow's Neural Network Playground.

Challenge

Neural network concepts are notoriously abstract. The challenge was to make each component definition concrete, visual, and connected to observable Playground behavior — not just a glossary of terms.

Solution

Designed an interactive SVG architecture diagram with hover tooltips per neuron, six detailed component cards each featuring a "Playground Insight" box, a three-step training cycle visualization, and a key insights summary tied directly to observed behavior.

HTML5 / SVG TensorFlow Playground Tailwind CSS Vanilla JS
Artifact 3

LLM Infrastructure: Training Process & Costs

View Artifact →

Context

The assignment required a comprehensive breakdown of how large language models are trained — from data acquisition through RLHF and deployment — and how costs (data, compute, energy, time) break down for real models like GPT-4, LLaMA-3, Gemini, and Claude.

Challenge

LLM training details are partially proprietary, spread across dozens of papers and industry reports (Epoch AI, SemiAnalysis, DeepMind), and inherently complex. The challenge was synthesizing this into a clear, credible, visually rich framework — not a wall of caveats.

Solution

Built a 7-stage clickable accordion pipeline, four cost-pillar cards with animated bar charts, a full model comparison table across 4 models, and a Chinchilla scaling laws section — all backed by 9 academic and industry references.

HTML5 Tailwind CSS Epoch AI / SemiAnalysis data GitHub Pages

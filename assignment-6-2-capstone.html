Why RAG — Not Just an LLM?

  • Patients need personalized, accurate answers drawn from their actual records — not generic text.
  • Pure LLMs hallucinate; in healthcare, a fabricated drug interaction answer can cause serious harm.
  • RAG retrieves verified clinical documents first, then generates a response grounded in that context.
  • Every response includes source citations — patients and clinicians can verify claims instantly.
  • Enables real-time updates: update the knowledge base without retraining the model.

Data Sources & Pipeline

Slide 07 · Data Design

Data Sources (by type)

  • Uploaded Files: 4,200 clinical care guides (PDF/DOCX) + 870 insurance policy documents — CMO-reviewed
  • Web APIs: Epic FHIR R4 (real-time EHR) · DrugBank API · NIH MedlinePlus (weekly sync)
  • Databases: PostgreSQL scheduling DB (~12M records) · Azure Cosmos DB (conversation logs)

Data Quality Controls

  • All clinical docs require CMO approval before indexing
  • Stale doc detector flags content older than 18 months
  • FHIR resources validated against HL7 schema on ingestion
  • Duplicate detection: cosine similarity ≥ 0.97 → deduplicate
  • Quarterly top-50 chunk audit by clinical QA team
Preprocessing Pipeline
1. Ingest

PDF → Textract OCR. APIs → normalized JSON. DB → REST microservice.

2. Clean

Strip headers/footers, normalize whitespace, fix encoding.

3. Chunk

512-token semantic windows, 64-token overlap, section-aware splits.

4. Enrich

Attach metadata: source, date, specialty, trust score, PII flag.

5. Embed

text-embedding-3-large → 3,072-dim vectors per chunk.

6. Index

Hybrid index: vector similarity + BM25 keyword in Azure AI Search.

All storage on Azure Government Cloud (US) · AES-256 at rest · TLS 1.3 in transit · 90-day conversation retention · HIPAA + HITECH compliant

Validation & Accuracy Demonstration

Slide 08 · Quality Assurance
🚫 Prompt Injection Shield

Azure AI Content Safety pre-screens all inputs. Adversarial patterns blocked before the LLM.

🩺 Clinical Boundary Enforcer

System prompt forbids diagnosis, prescriptions, or treatment decisions. Flagged responses redirect to provider.

⚠️ Emergency Keyword Detector

"Chest pain", "can't breathe", "suicidal" → immediate 911 guidance + nurse queue escalation.

🔒 PII Redaction (Presidio)

SSNs, DOBs, MRNs masked in every response before delivery to patient.

📌 Source Citation Enforcer

Ungrounded claims trigger fallback: "I don't have enough info — contact your care team."

📏 Response Length Limiter

Max 400 tokens per turn. Long answers chunked with "Continue?" for patient readability.
